# Knowledge (Generated)

Bu klasör agent’lar için **makine-okunur** bağlam çıktıları içerir.

## Güncelleme

- `node tools/indexer/index.js`

## Dosyalar

- `components.json`: Angular component sembolleri (dosya + selector/class).
- `services.json`: Angular service/provider sembolleri.
- `modules.json`: Angular NgModule sembolleri.
- `imports-graph.json`: TS import ilişkileri (from → to).
- `symbols.json`: TypeScript export/metod imzaları ve member metadata (deterministik AST taraması).
- `symbols.by-source.json`: sourceId bazlı dosya/exports sayısını sunar.
- `sources.json`: tsconfig tabanlı source tanımları ve include/exclude ipuçları.

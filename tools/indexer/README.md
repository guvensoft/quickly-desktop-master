# Indexer

Basit bir TypeScript tarayıcısı. Angular component/service/module sembollerini ve import grafını çıkarır.

## Run

- `node tools/indexer/index.js`

## Outputs

- `docs/knowledge/components.json`
- `docs/knowledge/services.json`
- `docs/knowledge/modules.json`
- `docs/knowledge/imports-graph.json`
